# Movie service

## Technology used

- Node
- Express 
- Typescript
- Rest API
- Jest

<br />

## How to start application
1. Clone repo
`git clone https://github.com/adamsmichal/Node_Express_Recruitment_Task.git`

2. Install dependencies
`npm install`

3. Start app
`npm run dev`

4. You can run tests
`npm run test`
